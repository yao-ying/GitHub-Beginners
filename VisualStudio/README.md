## 简介

无论是GitHub Desktop还是其他的GitHub客户端，都是基于git程序的一个GUI Shell。由于网络的原因，Github Desktop的下载往往很困难，所以VisualStudio或者其他的IDE之中内置的git客户端就成为了大部分人的选择。这里介绍了如何使用VisualStudio内置的Github官方客户端进行对GitHub上面的repository的操作。
